

					Flink


-Apache flink is a stream processing framework 

why Flink
---------
-Provides a high-throughput, low latency streaming engine
-Support for event time processing and state management
-Executes arbitrary dataflow programs in a data-parallel and pipelined
manner
-Flink's pipelined runtime system enables the execution of bulk/batch 
and stream processing programs
-Flink's runtime supports the execution of iterative algorithms
